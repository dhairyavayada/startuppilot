# StartupPilot (Appsmith)

This repository contains the open-sourced Appsmith applications built for the StartupPilot project.

## Contents
- Founder / MIS dashboard (StartupPilot Startup)
- Investor dashboard (StartupPilot Investor)

These applications were built using Appsmith and are shared for learning and reuse.

## How to use
1. Set up an Appsmith instance (Cloud or self-hosted)
2. Import the JSON files from the `/apps` directory
3. Create your own datasources to match the names used in the apps
4. Map queries to your own data

## Notes
- No credentials or real data are included
- You must configure your own datasources
- These apps are provided as-is
